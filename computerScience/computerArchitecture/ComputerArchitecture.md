# Computer Architecture

### 블랙박스

내부 동작 원리나 구조를 몰라도 입력(Input)을 넣으면 어떤 출력(Output)이 나오는지 관계만으로 시스템을 이해하는 개념

### 버스

각 장치들에서 작은 전선다발을 통해 데이터가 이동하는 통로

### 기계어 변환

사람이 작성한 코드를 컴퓨터가 해석할 수 있도록 기계어로 변환.

- 컴파일러
  사람이 작성한 프로그램을 실행하기 전에 전체 코드를 읽어서 컴퓨터가 이해할 수 있는 기계어로 변환.
  장점: 코드를 기계어로 변환하는 과정에서 발생한 에러를 미리 잡을 수 있다.
  단점: 실행 전 전체 코드를 기계어로 변환하는데 시간이 오래걸림

- 인터프리터
  프로그램을 실행하는 순간부터 코드를 한 줄씩 읽어가며 기계어로 변환하고 컴퓨터가 이를 즉시 실행하는 방법.
  장점: 코드를 한줄씩 해석하며 기계어로 변환하기 때문에 단계별로 확인하며 개발할 수 있음
  단점: 컴파일 언어보다 실행속도가 느림. 실행 중에 한줄 씩 해석하기 때문에 프로그램 실행 도중에 문법 오류가 발생할 수 있음

### CPU(Central Processing Unit)

중앙 처리 장치 구성 요소

- 프로그램 카운터(Program Counter)
- 제어 유닛(Control Unit): 명령을 해석하고 실행하도록 하는 장치. CPU 내에서 가장 복합한 부분
- 산술 논리 연산장치(ALU): 산술연산과 논리연산을 수행하는 여러 회로로 구성
- 레지스터: 컴퓨터의 가장 작은 단위의 메모리. 저장용량은 작지만 속도가 매우 빠름

### 메모리

메인 메모리(Main Memory)와 보조 메모리(Secondary Memory)로 나뉜다.

메인 메모리는 RAM(Random Access Memory)와 ROM(Read Only Memory)로 나뉜다.

- RAM: 휘발성 메모리. 전원이 끊기면 저장된 모든 내용이 사라짐.
  무작위 접근이 가능하여 어느 위치의 데이터든 동일한 속도로 접근할 수 있다.
- ROM: 데이터를 영구적으로 저장할 수 있는 비휘발성 메모리.
  한번 저장한 데이터를 수정할 수 없다. 주로 컴퓨터 부팅을 담당하는 BIOS를 저장하는 용도로 사용된다.

레지스터는 RAM보다 읽고 쓰는 속도가 훨씬 빠르다.
자주 사용하는 데이터는 가능한 한 레지스터에 보관하는 것이 효율적이다. CPU가 아무리 빠르게 동작해도 RAM에서 데이터를 가져오는 시간이 느리기 때문에 성능에 한계가 있다.
이를 해결하기 위해 CPU 내부에 더 빠른 메모리인 캐시 메모리를 둔다.
캐시메모리는 레지스터보다는 느리지만 RAM보다는 빠른 장치로 필요한 데이터를 미리 저장해두어 전체적인 성능을 향상 시켜준다.

### 주변 장치

주변장치는 크게 입출력 장치(I/O Device)와 보조 기억 장치(Secondary Memory)로 구분된다.

- 입출력 장치
  - 입력 장치(Input Device): 키보드, 마우스
  - 출력 장치(Output Device): 모니터, 스피커, 프린터

- 보조 기억 장치: 메인 메모리를 보조하는 장치. 주로 부족한 용량을 보조하는 역할. 속도는 상대적으로 느리지만 용량이 크다.
  - HDD(Hard Disk Drive): 내부의 디스크가 회전하며 데이터에 접근하므로 위치에 따라 접근속도가 다르다.
  - SSD(Solid State Drive): 전지적으로 데이터에 접근하여 하드디스크보다 빠른 속도를 제공한다.

### 32bit, 64bit 컴퓨터

32비트의 경우의 수는 약 42억개
64비트의 경우의 수는 약 18경개

컴퓨터가 계산을 할 때는 CPU 내 산술 논리연산 장치가 레지스터에 있는 값을 가져와 명령에 맞는 연산을 하고 결과는 다시 레지스터에 저장한다.
CPU에는 레지스터가 여러개 존재하는데 각 레지스터들은 1비트를 저장할 수 있는 플립플롭이라는 회로를 이용해 만든다.
만약 플립플롭 8개로 레지스터를 만드었다면 8비트 레지스터라고 부른다.
CPU 내 레지스터가 32비트 레지스터라면 산술 논리연산장치가 한 번에 계산하는 단위는 32비트가 되고 이를 32비트 CPU, 32비트 컴퓨터라고 부른다.
레지스터가 64비트라면 이 데이터를 옮기는 전선도 64개가 되고 이 버스도 64비트 버스라고 부른다.
